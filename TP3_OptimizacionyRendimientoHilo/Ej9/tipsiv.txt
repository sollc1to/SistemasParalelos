Para ver que tipo de instrucciones vectoriales utilizar, podemos realizar lo siguiente: 

-Avergiuar que SIMD soporta el procesador. ¿Que es SIMD? Single instruction - Multiple Data.
Esto lo podemos ver con el comando lscpu o cat /proc/cpuinfo | grep -i flags | head -n 1.

Ahi podemos buscar flags como: 
sse, sse2, ssse3, sse4_1, sse4_2

avx

avx2

fma

avx512f, avx512bw, avx512dq, etc.

-Pensar en el ancho vectorial. Es decir, con que tamaño vamos a trabajar, y en base a eso buscar la instruccion.

gcc -O3 -march=native -fopt-info-vec-optimized archivo.c -o prog

Este comando trata de vectorizar automaticamente, dando como salida que parte del programa vectorizo.
El nivel 03 de gcc vectoriza automaticamente.
