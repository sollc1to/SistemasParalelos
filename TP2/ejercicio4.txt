Salida del programa: 

Submitted batch job 163687.

En este caso estamos enviandole un script para ejecutar con batch.

-e: Dirige la salida de errores a un archivo.
--exclusive: Job allocation can not share nodes with other running
nodes.
-N: Request that a minimun of minnodes nodes be allocated to this job.
if only one number is specified, this is used as both the minimun and maximun node count.

-o: Salida en el .txt especificado.
 
--partition= Blade: Request a specific partition for the resource allocation.

#define _GNU_SOURCE // sched_getcpu(3) is glibc-specific
#include <stdio.h>
#include <unistd.h>
#include <sched.h>

int main( int argc, char *argv[]) {
   char hostname[128];
   gethostname(hostname, sizeof(hostname));

   printf("Ejecutando en nodo %s, core %d\n", hostname, sched_getcpu());
   return 0;
}
